# jwt-express-nodejs
A demo of JWT implementation using ExpressJS

# Install
After clone, just run 

    npm install

# Run 

    node index.js

You can also define secret hash and timeout for generated token

    export SECRET=SOSECRET && export EXPIRE_TIME=60 && node index.js
